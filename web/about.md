# 📖 关于本站

欢迎来到 **Code & Think**！

## 关于我们

这是一个专注于技术与思考的博客站点。在这里，你会看到：

- 💡 **技术文章** - 编程笔记、技术分享、最佳实践
- 📚 **算法题解** - 数据结构与算法的详细解析
- 🛠️ **开源工具** - 实用的开发工具和资源推荐
- ✨ **生活随笔** - 程序员的日常生活与思考

## 技术栈

本站使用以下技术构建：

- [VitePress](https://vitepress.dev/) - 基于 Vite 的静态网站生成器
- [Vue.js](https://vuejs.org/) - 渐进式 JavaScript 框架
- [Markdown](https://daringfireball.net/projects/markdown/) - 轻量级标记语言
